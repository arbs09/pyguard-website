Features

Leveling

The bot has a per-server leveling system, so you get rewarded levels on your discord server. (the bot has to be on that server too)

Global Leveling

For users, there is a global leveling feature, so you not only have a level per server, but also a level across all servers you (and the bot) are on.

manuall User Verify

PyGuard collects data about users' messaging behavior so you can verify if a user is trustworthy manually

Features Coming Soon

Automatic User Verify

Let the bot automatically check new users if they have a minimum global level / XP and verify them automatically, or if they don't have the minimum or aren't recorded in the global database, make the check a checkbox to verify them.

global user warn system

If a user gets banned on a server in the network of servers with PyGuard on it, a moderator of that server can send a report about the ban and other server moderators with PyGuard on their server get a warning message.
